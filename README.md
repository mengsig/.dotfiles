# My .dotfiles

Hereunder are my .dotfiles.

## Installation Guide
This follows you do a hyprland installation, and then you execute the following:

1. ```git clone https://github.com/mengsig/.dotfiles.git```
2. ```cd .dotfiles```
3. ```bash install.sh```
4. ```cd ~/.config/nvim/```
5. ```nvim .```
6. ```:PackerInstall```
7. ```:PackerSync```


#### There you are! Done... enjoy ( ;
